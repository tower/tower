Getting Setup

You've probably already got Tower.js installed if you're reading this, but just in case we've provided a little instruction. To get started with Tower.js you'll need the following software:

  • Node.js
  • Node Package Manager (aka, npm)

With those two installed you only need to do run (inside your terminal) these commands to create your first app:

  1. npm install -g design.io forever tower
  2. tower new app_name_here
  3. cd app_name_here
  4. npm install

Server: Go!

Once you've set up your application you can run the server with the forever library. Just use the below command in your terminal: 


  $ forever server.js

You should be able to open your browser to http://localhost:3000/ and see this page!
